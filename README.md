# NasaList

NasaList is an infinite scroll app, which contains a lot of pictures of the sky in each day of the year, and the app makes available in a infinite list

## Requirements

```
iOS 13+
Xcode 11
Swift 5
UIKit
```

## This class uses:

```
LottieFile (For loading animation)
Pagination Function
API Request
```
